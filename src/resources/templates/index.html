Instructions

Start RJZ Server on your computer
RJZ Server creates a small website on your computer that you can reach from your iThing to install scenes.
Select scene directory
In File -> Set scene directory you can point Rjzserver to where you keep your scenes.
Point it to a directory that contains scene folders with the .rj file name extension.
Install scenes
To install scenes, you browse to this page with Safari on your iThing by putting 'http://${headers['HOST']}/' into the address bar of Safari on your device.

Tap the install button on a scene, and RjDj will open and download the scene from your computer.
